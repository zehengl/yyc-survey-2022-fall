A study on the Fall Survey of Calgarians (2022) data

## Environment

- Python 3.9

## Getting Started

    python -m venv .venv
    .venv\Scripts\activate
    python -m pip install -U pip
    pip install -r requirements.txt

> Use `pip install -r requirements-dev.txt` for development and docs.

## Docs

    mkdocs serve
